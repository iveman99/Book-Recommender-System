About This Project

A neon-powered, ML-driven book recommender system that understands what readers love — built with precision, passion, and data intelligence.

What is iVeman Recommender?

This project blends machine learning with clean UI to help readers discover new books they are likely to enjoy. It learns patterns from real rating behaviour and uses similarity mathematics to deliver smart suggestions instantly.

Dataset · The Foundation

We begin with a dataset containing thousands of readers, ratings, and books. But raw data is noisy — so we clean, filter, and refine it until only meaningful signals remain.

How the ML Model Works

1
Data Prep: Filter books with enough ratings for meaningful patterns.
2
User–Book Matrix: Convert ratings into a grid that represents books as vectors.
3
Cosine Similarity: Measure how closely two books relate based on user behaviour.
4
Top Matches: Rank and return the most similar books.
5
Flask Serving: Backend sends results to UI instantly.

How to Use This System
